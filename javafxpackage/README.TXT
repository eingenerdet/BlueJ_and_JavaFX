PROJEKTBEZEICHNUNG: JavaFX Applikation
PROJEKTZWECK: Demonstration
VERSION oder DATUM: 29.12.2019
WIE IST DAS PROJEKT ZU STARTEN: View > Run JavaFX Application
AUTOR(EN): Markus Tank
BENUTZERHINWEISE: Das Projekt dient der Demonstration von JavaFX in BlueJ. 
Es werden ein Textfeld, ein Button und ein Label in einer GridPane erzeugt.
Gibt man beliebigen Text in das Textfeld ein und drück daraufhin den Button, 
so wird dieser Text vom Label angezeigt.
